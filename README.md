# COAP Exercise

## Python Version Required
You **must use Python 3.11** for this exercise.  
Other versions (e.g., 3.12) are **not supported** and will cause the servers to fail.

---

## 1. Download the Repository

```sh
git clone https://github.com/clamar92/coap-exercise.git
```

Then open the project folder in **Visual Studio Code**.

---

## 2. Create and Activate the Virtual Environment

1. Create the virtual environment using Visual Studio Code (make sure it uses **Python 3.11**).
2. Activate the virtual environment.

---

## 3. Install Required Packages

```sh
pip install -r requirements.txt
```

---

## 4. Start the Servers

```sh
python run_servers.py
```
